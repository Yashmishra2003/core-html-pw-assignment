1. Head tag- The head tag is used for holding Meta information, title, links, etc.
  2. Br tag-
    The br tag inserts a single line break.
  3. Title tag- The title tag defines the title of the document
  4. Div tag-
    The Div tag defines a division or a section in an HTML document
  5. P tag-

    The paragraph or the 'P' tag in HTML helps users define a paragraph
  6. Style tag- The style tag is used to define style information (CSS) for a document.
  7. Body Tag- The body tag in HTML is used to define the main content present inside an HTML page
  8. U tag- The u tag in HTML stands for underline, and it's used to underline the text enclosed within the tag
  9. Sub tag- The sub tag defines subscript text Subscript text appears half a character below the normal line
  10. Sup tag- The sup tag is used to display a text as superscript in HTML.
